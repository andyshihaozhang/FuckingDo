# FuckingDo

![FuckingDo Logo](Assets/Logo.png)

一个现代化的 WPF 任务管理应用程序，帮助你快速高效地完成工作。

## 功能特性

- 现代化的用户界面设计
- 任务管理和追踪
- 多语言支持
- 系统托盘集成
- 语法高亮支持
- 依赖注入支持

## 技术栈

- .NET 8.0
- WPF
- WPF-UI 4.0.2
- CommunityToolkit.Mvvm 8.4.0
- Microsoft.Extensions.Hosting 9.0.4
- Lepo.i18n 2.0.0

## 系统要求

- Windows 操作系统
- .NET 8.0 运行时

## 安装说明

1. 下载最新版本的可执行文件
2. 双击运行安装程序
3. 按照安装向导完成安装

## 使用说明

1. 启动应用程序
2. 在系统托盘中找到应用图标
3. 点击图标打开主界面
4. 开始管理你的任务

## 开发环境设置

1. 克隆仓库
```bash
git clone [repository-url]
```

2. 使用 Visual Studio 2022 或更高版本打开解决方案
3. 还原 NuGet 包
4. 构建并运行项目
